# linux

## 文件相关

建立软连接

```shell
ln -s /root/node-v16.13.1-linux-x64/bin/node /usr/bin/node
ln -s /root/node-v16.13.1-linux-x64/bin/npm /usr/bin/npm
```

取消软连接

```shell
rm -rf /usr/local/bin/node
```

用户名密码连接

```shell
ssh -t 用户名@ip -p 端口号
```

产看指定端口的进程

```shell
sudo lsof -i :3000
```

杀死进程

```shell
kill -9 {pid}
```
