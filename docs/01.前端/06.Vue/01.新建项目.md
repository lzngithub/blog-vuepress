# 新建项目（vue3， @vue/cli 5，mac， yarn）

## 参考官网

vue3官网：[https://v3.cn.vuejs.org/](https://v3.cn.vuejs.org/)   
@vue/cli官网：[https://cli.vuejs.org/zh/](https://cli.vuejs.org/zh/)

## 基本命令

适用vue脚手架生成项目

安装脚手架

```shell
sudo yarn global add @vue/cli
```

创建项目

```shell
vue create vue3-practice
```
